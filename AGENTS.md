# AGENTS.md

## 作用范围

- 本文件对整个仓库生效。
- 若子目录存在更近的 `AGENTS.md`，则子目录文件优先于本文件。

## 语言规则

- 默认使用中文进行沟通、说明、注释与文档编写。
- 新增或更新发布文档（`docs/releases/**`）默认使用中文。
- 前端界面文案默认使用中文。
- 仅当用户明确要求英文时，才切换为英文输出或英文文案。

## 产品与界面规则

- 当前阶段没有多会话能力时，侧栏仅保留单条固定会话占位。
- 当前阶段没有账号体系时，用户区使用占位信息，不引入真实账户逻辑。
- 优先复用仓库中的最新品牌资产；当用户指定 logo 文件名时，按指定文件使用。

## 编辑规则

- 优先做最小改动，避免无关重构。
- 在不影响需求的前提下，保持向后兼容与现有功能稳定。
- 修改 UI 时，确保桌面端与移动端可用。
