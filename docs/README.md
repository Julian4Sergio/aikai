# aikai 文档基线

本目录用于沉淀项目的规划与研发基线文档，后续开发默认遵循本目录规范。

## 目录分层

- `docs/foundation/`：基础规范
- `docs/templates/`：模板库
- `docs/releases/v1.0/`：`V1.0` 版本文档包
- `docs/releases/v1.1/`：`V1.1` 增量范围与承接项

## 基础规范（foundation）

- `docs/foundation/PRD-v1.md`：产品需求与版本范围（`V1.0` 到 `V2.0`）
- `docs/foundation/ROADMAP.md`：版本里程碑与验收标准
- `docs/foundation/TECH_STACK.md`：技术栈基线
- `docs/foundation/ARCHITECTURE_GUARDRAILS.md`：架构约束与升级边界
- `docs/foundation/DEV_WORKFLOW.md`：开发流程与交付规范
- `docs/foundation/VERSION_DELIVERY_STANDARD.md`：每版统一交付标准与门禁

## 模板库（templates）

- `docs/templates/VERSION_DOC_TEMPLATES.md`：版本文档 10 类模板
